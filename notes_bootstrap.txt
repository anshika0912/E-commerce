Grid

Need of grid in designing UI

  bootstrap provide 12 colimn grid structure i.e. browser width id divided into 12 equal space column

Margin
    mb/pb - for classes that set margin-bottom or padding-bottom
	mt/pt - margin-top
	ms - (start) for classes that set margin-left or padding-left in LTR, margin-right or padding-right in RTL
	mx - for classes that set both *-left and *-right
	ml - for classes that set margin-left or padding-left
	me - (end) for classes that set margin-right or padding-right in LTR, margin-left or padding-left in RTL
    mt -  for classes that set margin-top or padding-top
	my - for classes that set both *-top and *-bottom
md - medium devices
